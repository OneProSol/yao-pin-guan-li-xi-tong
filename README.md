### 一、相关技术
- 前端：`thymeleaf`、`HTML` / `CSS` / `Javascript`。
- 后端：`Springboot`等。
- 数据库：`MySQL`

### 二、开发软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
### 三、部署所需环境

- 后端：`jdk1.8`。
- 数据库：`mysql5.7以上`

### 四、功能描述及功能图展示
系统分为：管理员。

管理员功能：1. 登录、2.出入库信息、3. 问题药品记录、4. 药品保质期检查、5. 销售记录、6. 退货记录、7. 药品信息管理、8. 供应商管理、9. 退货供应商列表
